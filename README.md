# 💬 Fórum Hub — Challenge
## 📖 Sobre o Projeto

O Fórum Hub é uma API REST desenvolvida em Java com Spring Boot que simula um fórum de discussões.
A aplicação permite criar, listar, atualizar e excluir tópicos, além de possuir autenticação e autorização com JWT para proteger os endpoints.

Os dados são persistidos em banco de dados relacional, garantindo armazenamento e consulta das informações.

## 🚀 Funcionalidades

📝 Criar tópico

📚 Listar tópicos

🔍 Detalhar tópico

✏️ Atualizar tópico

❌ Excluir tópico

🔐 Autenticação com JWT

## 🛠️ Tecnologias Utilizadas

Java 17

Spring Boot

Spring Web

Spring Data JPA

Spring Security

JWT

PostgreSQL

Flyway

Maven


## 🎯 Objetivo do Projeto

Este projeto foi desenvolvido como parte de um Challenge da Alura para praticar:

Desenvolvimento de APIs REST

Autenticação com JWT

Persistência de dados com JPA

Boas práticas de estruturação de projetos Spring Boot
